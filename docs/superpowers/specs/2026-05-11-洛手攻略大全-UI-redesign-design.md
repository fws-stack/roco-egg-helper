# 洛手攻略大全 — UI 整体重设计规格

日期: 2026-05-11
状态: 已确认，待实现

## 设计目标

将微信小程序「洛手攻略大全」从当前工具型风格，重新设计为**杂志/图鉴风**——优雅、耐看、有翻阅典籍的仪式感。

## 色彩系统（暖色羊皮纸）

| 角色 | 色值 | 用途 |
|------|------|------|
| 页面底色 | `#FBF8F0` | 全局背景 |
| 卡片底色 | `#FFFFFF` | 卡片、列表项 |
| 金色点缀 | `#C9A96E` | 分割线、箭头、激活态边框、标题装饰线 |
| 主文字 | `#5D4E37` | 标题、正文 |
| 辅助文字 | `#8B7355` | 描述、标签 |
| 浅文字 | `#B8A890` / `#C9B99E` | 次要信息、placeholder |
| 绿色强调 | `#4A6741` | 精灵名、草系等自然元素 |
| 暖色强调 | `#D4A574` | 异色角标、特殊标记 |
| 边框 | `#E8DCC8` / `#EDE5D5` | 卡片边框、分割 |
| 卡片阴影 | `rgba(139,115,85,0.06)` | 轻微投影 |

## 字体

微信小程序仅支持系统字体，不做自定义字体配置。

- 标题字体族: `Georgia, "Times New Roman", serif`（Serif 标题营造典籍感）
- 正文字体族: `-apple-system, "PingFang SC", "Helvetica Neue", sans-serif`（保持可读性）
- POKEDEX 标签: 小号字 + 宽字间距 (`letter-spacing: 3-4rpx`)，大写英文

## 导航结构

### TabBar（3 Tab）

| Tab | 页面 | 图标 |
|-----|------|------|
| 首页 | `pages/index/index` | 🏠 |
| 图鉴 | `pages/pokedex/pokedex` | 📖 |

- 首页 → 图鉴（switchTab）
- 首页 → 更多（switchTab）
- 图鉴 → 精灵详情（navigateTo）
- 更多 → 蛋参数查询（navigateTo）
- 更多 → 后续功能入口

## 页面重设计

### 1. 首页 (`pages/index/index`)

**布局**: 品牌头图 + 功能入口卡片 + 最近查询

**头图区域**:
```
POKEDEX（小字宽间距，金色）
精灵图鉴（Georgia 大标题，深棕）
────（金色短分割线）
```

**入口卡片**: 两个主卡片，采用白色圆角卡片 + 金色右边距装饰：
- 精灵图鉴卡片: 📖 图标 / "浏览347只图鉴精灵与进化链"
- 蛋参数查询卡片: 🥚 图标 / "输入蛋的长和重判断精灵"

卡片样式: 白底 `#FFFFFF`，边框 `#E8DCC8`，圆角 `12rpx`，轻阴影。点击态背景微变。

**最近查询区块**: 同上风格，仅保留蛋类型历史。

### 2. 图鉴页 (`pages/pokedex/pokedex`)

**从列表型改为双列网格型**。

**搜索栏**: 白色圆角搜索框，边框 `#E8DCC8`，placeholder 颜色 `#C9B99E`。

**属性筛选**: 横向滚动的胶囊标签。选中态：`#C9A96E` 背景金色，白色文字；未选中：白色背景，`#8B7355` 边框+文字。

**网格卡片**:
- 2 列布局，`gap: 12rpx`
- 每张卡片: 白色背景，圆角 `10rpx`，轻阴影
- 内容：精灵头像（圆形、浅色背景）、精灵名（粗体 `#5D4E37`）、序号+系别（小字 `#B8A890`）
- 有异色的精灵: 右上角 `#D4A574` 小角标

**区域切换**: 顶部 Tab 切换（全部/风眠省/洛克里安），风格同属性筛选但更突出。

**底部计数**: "共 347 只" 小字居中，`#C9B99E`。

### 3. 精灵详情页 (`pages/sprite-detail/sprite-detail`)

保持当前页面结构，重新配色：
- 头图区域：杂志式大图 + 编号 + 名称（Georgia 字体）
- 属性标签：暖色半透明底色
- 进化链组件：重新设计（见组件部分）
- 基本信息：卡片式布局，使用金色分割线

### 4. 蛋参数查询页 (`pages/egg-query/egg-query`)

保持功能不变，重新配色：
- 输入区：白色卡片，边框 `#E8DCC8`
- 滑块/输入框：激活色改为 `#C9A96E`
- 结果展示区：同新配色

### 5. 更多页 (`pages/more/more`) — 新建

- 功能入口列表（蛋参数查询等）
- 关于信息
- 样式同首页卡片风格

## 组件重设计

### sprite-card

- 当前: 游戏风格卡片
- 新设计: 杂志风格，圆形头像 + Georgia 字体名称 + 细金色边框
- 支持 `size` 属性（small / normal / large），不同尺寸不同展示密度

### evolution-chain

- 当前: 横向排列
- 新设计: 竖向排列为主（大图模式下可横向）
- 每个阶段: 圆形头像 + 等级标注 + 金色箭头 → 连接
- 当前阶段高亮: 金色边框 + 稍大尺寸

### egg-result

- 重新配色：白色卡片 + 金色点缀
- 精灵列表项: 保持可点击跳转

### empty-state

- 插图风格: 线性简笔画风
- 文字: 辅助色

## 全局样式变量
